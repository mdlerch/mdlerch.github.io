Setting up vim for Rcpp

Motivation

My motivation for writing this post is so to have a reference for myself about how I set up vim for Rcpp. If anyone else finds this useful, that's just bonus! Though the fine details are Rcpp related it should be applicable for general .cpp stuff.

Plug-ins

The three vim plug-ins that I'll need are:

  1. neomake, to check for syntax errors. Note that this is a neovim plug-in.
  2. YouCompleteMe, which provides intelligent auto-completion, and
  3. TagHighlight, which provides intelligent syntax highlighting.
